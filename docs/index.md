## What's in the Box?

=== "Complete Kit"

    All DryBase components included. Only the housing needs to be 3D-printed by you.

    - 150W Power Supply Unit (PSU)
    - C5 1.5m power cord (region-specific)
    - 3D-printed heating unit parts
    - 3.5-inch capacitive touch display
    - Cables
    - Drybox parts (both printed and non-printed)
    - Main board, Power board, Sensor board
    - Heating element
    - Exhaust fan, Blower fan
    - Adhesives bag, Screws / fasteners bag, Spares bag

=== "Core Kit"

    All non-printed components included. All 3D-printed parts must be printed by you.

    - 150W Power Supply Unit (PSU)
    - C5 1.5m power cord (region-specific)
    - 3.5-inch capacitive touch display
    - Cables
    - Drybox parts (non-printed only)
    - Main board, Power board, Sensor board
    - Heating element
    - Exhaust fan, Blower fan
    - Adhesives bag, Screws / fasteners bag, Spares bag
