# Filametric Studio

Filametric Studio is a free desktop application to monitor, control and manage your DryBase over Wi-Fi.

**Local. Free. No subscriptions.**

## Features

- **Real-time monitoring** — View humidity, temperature, and drying progress.
- **Filament profiles** — Manage drying profiles for different filament types.
- **Device settings** — Configure sounds, display, temperature units, and more.
- **Firmware updates** — Update your DryBase firmware directly from the app.

## System Requirements

| Platform | Requirement |
|----------|-------------|
| Windows | Windows 10 (64-bit) or above |
| macOS | macOS 10.15 or above (coming soon) |

## Getting Started

1. Download and install Filametric Studio.
2. Connect your DryBase to your Wi-Fi network via the touchscreen.
3. Open Filametric Studio — your DryBase should appear automatically.
4. Start monitoring and controlling your dryer.

No account, no cloud, no internet required — everything runs locally on your network.
