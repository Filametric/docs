## Installation

=== "Windows"

    1. Download the `.exe` installer.
    2. Run the installer and follow the prompts.
    3. Launch Filametric Studio from the Start menu or desktop shortcut.

    !!! note "Windows SmartScreen"
        On first launch, Windows may show a SmartScreen warning. Click "More info" → "Run anyway". The installer is signed by Filametric.
