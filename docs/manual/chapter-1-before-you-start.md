## Safety Guidelines

!!! warning "Electrical Safety"
    The DryBase uses mains electricity. Never work on electrical connections while the unit is plugged in. Always double-check wiring before powering on for the first time.

!!! warning "High Temperature"
    The DryBase operates at temperatures up to 100°C. Allow the unit to cool completely before handling internal components after operation.
