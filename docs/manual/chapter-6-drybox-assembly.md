# Chapter 6 — Drybox Assembly

In this chapter, you will set up the drybox using a compatible cereal container (Wildone or GoMaihe).

!!! note "Compatible containers"
    The DryBase is designed to work with Wildone and GoMaihe cereal containers (~4L, 6cm lid opening). Other containers may work but are not officially supported.
